# Watermarking

## Language
Python3.7

## Requirements
* Your computer should have been installed some package
  * numpy
  * pillow
  
  ```
  pip3.7 install numpy
  ```
  ```
  pip3.7 install pillow
  ```

## Execution
```
python3.7 Hw1.py input.png watermark.png
```

## Result
Save in the ```result``` directory
